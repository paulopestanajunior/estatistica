# Plano 1 — Estatística para Data Science

Plano de estudos focado em estatística pesada para Data Science, com base no conteúdo da pós e prática aplicada em Python.

## Visão geral

- duração sugerida: 6 semanas
- carga: 3h por dia, 6 dias por semana
- foco: teoria + interpretação + prática em Python
- bibliografia central: Morettin, Ross, Bolfarine, Mingoti
- bibliografia aplicada: Data Science para Negócios, Hands-On Machine Learning
